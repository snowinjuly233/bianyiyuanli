# Compiler Lab Final Submission

This directory collects the final deliverables for the compiler lab.

- specs/: final .l / .y specifications
- src/: generator, semantic, IR, and tool source code
- generated/: generated scanner/parser source files
- tests/: regression test reports
- output/: AST / IR / basic block showcase outputs
- docs/: component reports and the final report
- ppt/: defense deck, preview images, and quality report
